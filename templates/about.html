About This Portal

This portal was developed by the E-Commerce App Team to improve efficiency and transparency in our product update workflow. It provides authorized employees with quick, centralized access to Manual Integration (MI) files that are now generated automatically.

What This Website Does

The system collects product data from our internal sources, prepares it in the correct MI format, and uploads it to the Suppy dashboard. Every run is logged for visibility, and all generated files are stored here for easy download and review.

  • Automatically generates MI files from our product database
  • Uploads files to Suppy for partner synchronization
  • Archives every file for future reference
  • Displays recent activity logs for accountability

The Process

1. Data is pulled from our product source (Google Sheets, ERP, etc.).
2. The automation formats the data into the Suppy MI CSV format.
3. The file is uploaded to Suppy automatically.
4. A copy is stored in the “Files” section for employee access.
5. All activity is recorded in the “Overview” log.
